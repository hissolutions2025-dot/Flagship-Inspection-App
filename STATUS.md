## 🏗️ App Architecture

- **Central state object:** `A`
- **Check status codes:** `"p"` = pass, `"f"` = fail, `"n"` = not applicable
- **Priority codes:** `"C"` = critical, `"H"` = high, `"M"` = medium, `"L"` = low
- **Photos:** `A.photos[itemId]` → `{ url, cap, kind: "I"|"R" }` (Issue / Rectification)
- **Checklist:** 553 SANS 10142-1 items across 16 sections
- **Three-phase filtering:** via `item.p3`
- **Storage:** localStorage (no backend)
